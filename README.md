# Customer Sales & Revenue Analysis

## Project Overview
This project analyzes customer sales data to identify revenue trends, top customers, and monthly performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Analysis
- Monthly sales trend analysis
- Customer-wise revenue contribution
- Data visualization for business insights

## Insights
- Sales increased in later months
- Repeat customers contributed higher revenue
- Visualization helped identify trends quickly

## Note
This project uses a practice dataset for learning purposes.
