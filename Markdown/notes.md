# Markdown Tutorial

### Intoduction

Markdown is a lightweight markup language that uses simple, plane-text formatting to create structured documents that can easily converted to HTML

Created by John Gruber in 2004, its design goal is to be human-readable and writeable, making it is easy to add formatting like heading, lists, and blod text using basic symbols.

Markdown is widley used for creating technical documentation, web content, readme files, and content for platfoem like githube, Azure DevOps, and online forms.

### How to create and preview MArkdown file?
- Create a new file and save it with a .md extension
- Open file in vs code 
- Press CTRL+K V (Windows/Linux), Press Cmd+K V (macOS)

```markdown

```

### Formatting text

```markdown

*This is also making text italic*

```
